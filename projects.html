Motor Control Panel

Below is a project that involved designing the PLC (Programmable Logic Controller) code and HMI (Human Machine Interface) touchscreen for a workshop test panel.

The panel included 8 motor drives that could be ran individually in manual, or according to preset automatic modes.

When any of the drives were clicked on from the screen above, a more detailed display would show data for each drive.

The same screen would generate for each group of drives, however different parameter lists would populate the display as appropriate depending on the drive chosen.

Pressing the 'burger menu' icon in the top right corner would bring up the navigation menu from any screen within the interface - improving usability.

Box Sorter Simulation

Here is a simulation project for a system that could detect different size boxes in an automated warehouse environment.

I created and built the scenario with virtual equipment in simulation software, as can be seen below.

I then linked the simulation to PLC programming software, where the code for the automation was created.

The HMI interface can be seen below, which offers full control over the system, offering automatic and manual modes.

When compared to the plan view below, the HMI layout mirrors the equipment accurately, offering a potential end-user a clear and intuitive experience.

PLC Code

Below is an example of ladder logic, a section of graphical code written by myself for an automated system.

It uses symbols and principles similar to schematic drawings but has a logical top-to-bottom layout.

When ladder logic is converted to structured text (also used with PLCs), similarities between other coding languages such as Python can be seen.

Teaching

During my time as a teacher, I created my own teaching materials that would cater to different learning styles

The logo below was used on a previous website of mine, marketing my services as teacher and performing musician.

I created the tuition book seen here, which offered a progression towards graded music exams. The book included access to self-written backing tracks available to download from my website.

Helpful visual diagrams and notation keys were included at the front of the book.

The colour-coded system I built below assisted younger drummers to play while reading music. The colours really helped to engage smaller children, but could be really useful for older pupils as well!
